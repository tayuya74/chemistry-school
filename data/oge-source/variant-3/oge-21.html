Задание 21

Химические свойства простых и сложных веществ

Дана схема превращений:

Mg → MgO (+ SO3) → X → Mg(NO3)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 3, 2025)

Одна из возможных цепочек: X = MgSO4.

  1. 2Mg + O2 = 2MgO
  2. MgO + SO3 = MgSO4
  3. MgSO4 + Ba(NO3)2 = Mg(NO3)2 + BaSO4↓

Допустимы эквивалентные переходы, например, растворение MgSO4 или MgO в азотной кислоте с последующим упариванием кристаллами Mg(NO3)2, если они согласуются с условием.
