Задание 22

Расчётная задача. Массовая доля, количество вещества, объём и масса

Задание 22 № 1022

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

К 340 г раствора с массовой долей нитрата серебра 6 % добавили избыток раствора хлорида железа(III). Вычислите массу образовавшегося осадка.

В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа

Уравнение:

3AgNO3 + FeCl3 = 3AgCl↓ + Fe(NO3)3

Вычисления:

  • m(AgNO3) = m(р-ра) · ω / 100 = 340 · 0,06 = 20,4 г
  • n(AgNO3) = m / M = 20,4 / 170 = 0,12 моль
  • по уравнению n(AgCl) = n(AgNO3) = 0,12 моль
  • m(AgCl) = n · M = 0,12 · 143,5 = 17,22 г ≈ 17,2 г (в бланке — по правилам округления КИМ)

В официальном ключе приведена масса AgCl = 17,22 г; при сдаче ориентируйтесь на требования бланка.

Задание 22 № 1045

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

Рассчитайте объём (н. у.) углекислого газа, который выделится при действии избытка карбоната магния на 97,33 г 15 % раствора соляной кислоты.

В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 1, 2025)

Уравнение:

MgCO3 + 2HCl = MgCl2 + H2O + CO2↑

Вычисления:

  • m(HCl) = m(р-ра) · ω / 100 % = 97,33 · 15 / 100 = 14,6 г
  • n(HCl) = m / M = 14,6 / 36,5 = 0,4 моль
  • по уравнению n(CO2) = ½ n(HCl) = 0,2 моль
  • V(CO2) = n · Vm = 0,2 · 22,4 = 4,48 л (н. у.)

Задание 22 № 1091

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

К 250 г раствора карбоната калия добавляли раствор нитрата магния до прекращения выпадения осадка. Масса осадка составила 16,8 г. Вычислите массовую долю карбоната калия в исходном растворе. В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 2, 2025)

Уравнение:

K2CO3 + Mg(NO3)2 = MgCO3↓ + 2KNO3

Вычисления:

  • n(MgCO3) = m / M = 16,8 / 84 = 0,2 моль
  • по уравнению n(K2CO3) = n(MgCO3) = 0,2 моль
  • m(K2CO3) = n · M = 0,2 · 138 = 27,6 г
  • ω(K2CO3) = m / m(р-ра) = 27,6 / 250 = 0,1104 → или 11 % (ключ издания; по точным данным получается ≈11,04 %)

Задание 22 № 1114

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

110 г раствора гидроксида калия смешали с избытком раствора нитрата железа(II). Выпал осадок массой 22,5 г. Вычислите массовую долю гидроксида калия в исходном растворе. В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 3, 2025)

Уравнение:

Fe(NO3)2 + 2KOH = Fe(OH)2↓ + 2KNO3

Вычисления:

  • M(Fe(OH)2) = 56 + 2 · (16 + 1) = 90 г/моль
  • n(Fe(OH)2) = m / M = 22,5 / 90 = 0,25 моль
  • по коэффициентам реакции n(KOH) = 2 · n(Fe(OH)2) = 0,50 моль
  • m(KOH) = n · M = 0,50 · 56 = 28 г (принято M(KOH) = 39 + 16 + 1)
  • ω(KOH) = m / m(р-ра) = 28 / 110 = 0,255 (≈ 25,5 %)

Задание 22 № 1137

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

Рассчитайте объём (н. у.) сернистого газа, который выделится при взаимодействии 146 г 20%-ного раствора соляной кислоты с избытком раствора сульфита калия. В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 4, 2025)

Уравнение:

K2SO3 + 2HCl = 2KCl + H2O + SO2↑

  • m(HCl) = m(р-ра) · ω = 146 · 0,20 = 29,2 г
  • n(HCl) = m / M = 29,2 / 36,5 = 0,80 моль

Задание 22 № 1160

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

К 300 г раствора гидроксида натрия добавили избыток раствора сульфата меди(II), при этом образовался осадок массой 73,5 г. Вычислите массовую долю гидроксида натрия в исходном растворе. В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 5, 2025)

Уравнение:

2NaOH + CuSO4 = Cu(OH)2↓ + Na2SO4

Вычисления (по сборнику):

  • n(Cu(OH)2) = m / M = 73,5 / 98 = 0,75 моль
  • по уравнению n(NaOH) = 2 · n(Cu(OH)2) = 1,5 моль
  • m(NaOH) = n · M = 1,5 · 40 = 60 г
  • ω(NaOH) = m(NaOH) / m(р‑ра) = 60 / 300 = 0,20 (20%)

Задание 22 № 1183

К избытку раствора хлорида алюминия добавили 16,8 г раствора с массовой долей гидроксида калия 10 %. Определите массу выпавшего осадка.

В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 6, 2025)

AlCl3 + 3KOH = Al(OH)3↓ + 3KCl

m(KOH) = ω·m(р‑ра)/100 = 16,8·10/100 = 1,68 г; n(KOH) = 1,68/56 = 0,03 моль; n(Al(OH)3) = ⅓·n(KOH) = 0,01 моль; m(Al(OH)3) = 0,01·78 = 0,78 г.

Задание 22 № 1068

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

К 150 г раствора карбоната натрия добавляли раствор хлорида кальция до прекращения выделения осадка. Масса осадка составила 12,0 г. Вычислите массовую долю карбоната натрия в исходном растворе.

В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа

Уравнение:

Na2CO3 + CaCl2 = CaCO3↓ + 2NaCl

Вычисления:

  • n(CaCO3) = m / M = 12 / 100 = 0,12 моль
  • по уравнению реакции n(Na2CO3) = n(CaCO3) = 0,12 моль
  • m(Na2CO3) = n · M = 0,12 · 106 = 12,72 г
  • ω(Na2CO3) = m(Na2CO3) · 100 % / m(р-ра) = 12,72 · 100 % / 150 = 8,48 %
