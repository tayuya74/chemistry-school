Задание 21 № 1021

Дана схема превращений:

Zn(NO3)2 →t° X → ZnSO4 → Zn(OH)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа
  1. 2Zn(NO3)2 →t° 2ZnO + 4NO2 + O2   (X — ZnO)
  2. ZnO + H2SO4 = ZnSO4 + H2O
  3. ZnSO4 + 2NaOH = Zn(OH)2 + Na2SO4

Допускается иной разумный путь к ZnSO4 от оксида цинка.

Задание 21 № 1044

Дана схема превращений:

Cu(NO3)2 → Cu → X → Cu(OH)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 1, 2025)
  1. Cu(NO3)2 + Fe = Fe(NO3)2 + Cu
  2. Cu + 2AgNO3 = 2Ag + Cu(NO3)2  (X — AgNO3)
  3. Cu(NO3)2 + 2NaOH = Cu(OH)2↓ + 2NaNO3

Задание 21 № 1090

Дана схема превращений:

Al → X → AlCl3 → Al(NO3)3

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 2, 2025)

В качестве X удобно взять оксид алюминия Al2O3.

  1. 4Al + 3O2 = 2Al2O3
  2. Al2O3 + 6HCl = 2AlCl3 + 3H2O
  3. AlCl3 + 3AgNO3 = Al(NO3)3 + 3AgCl↓

Допустимы и другие цепочки, согласованные с условием (например, получение AlCl3 через реакцию Al с хлором или HCl и последующий переход к нитрату).

Задание 21 № 1067

Дана схема превращений:

Fe(NO3)2 → Fe(OH)2 → X → Fe2O3

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа
  1. Fe(NO3)2 + 2NaOH = Fe(OH)2 + 2NaNO3
  2. 4Fe(OH)2 + 2H2O + O2 = 4Fe(OH)3  (X — Fe(OH)3)
  3. 2Fe(OH)3 →t° Fe2O3 + 3H2O
