Задание 21 № 1021

Дана схема превращений:

Zn(NO3)2 →t° X → ZnSO4 → Zn(OH)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (демоверсия ОГЭ 2026)
  1. 2Zn(NO3)2 →t° 2ZnO + 4NO2 + O2   (X — ZnO)
  2. ZnO + H2SO4 = ZnSO4 + H2O
  3. ZnSO4 + 2NaOH = Zn(OH)2 + Na2SO4

Допускается иной разумный путь к ZnSO4 от оксида цинка.

Задание 21 № 1044

Cu(NO3)2 → Cu → CuCl2 → Cu(OH)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

  3. CuCl2 + 2NaOH = Cu(OH)2↓ + 2NaCl

Задание 21 № 1090

Дана схема превращений:

Al → X → AlCl3 → Al(NO3)3

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 2, 2025)

В качестве X удобно взять оксид алюминия Al2O3.

  1. 4Al + 3O2 = 2Al2O3
  2. Al2O3 + 6HCl = 2AlCl3 + 3H2O
  3. AlCl3 + 3AgNO3 = Al(NO3)3 + 3AgCl↓

Допустимы и другие цепочки, согласованные с условием (например, получение AlCl3 через реакцию Al с хлором или HCl и последующий переход к нитрату).

Задание 21 № 1113

Дана схема превращений:

Mg → MgO (+ SO3) → X → Mg(NO3)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 3, 2025)

Одна из возможных цепочек: X = MgSO4.

  1. 2Mg + O2 = 2MgO
  2. MgO + SO3 = MgSO4
  3. MgSO4 + Ba(NO3)2 = Mg(NO3)2 + BaSO4↓

Допустимы эквивалентные переходы, например, растворение MgSO4 или MgO в азотной кислоте с последующим упариванием кристаллами Mg(NO3)2, если они согласуются с условием.

Задание 21 № 1136

Дана схема превращений:

Si → SiO2 → X → H2SiO3

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 4, 2025)

Одна из возможных цепочек: X = Na2SiO3.

  1. Si + O2 = SiO2
  2. SiO2 + 2NaOH = Na2SiO3 + H2O (сплавление или сильный нагрев с концентрированным раствором)
  3. Na2SiO3 + 2HCl = H2SiO3 ↓ + 2NaCl

Допустимы эквивалентные переходы (например, через силикат кальция и CO2 в водной среде), если они согласуются с условием.

Задание 21 № 1159

Дана схема превращений:

Na → X → NaNO3 → NaNO2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 5, 2025)

X = NaOH (ряд возможных превращений):

  1. 2Na + 2H2O = 2NaOH + H2↑
  2. NaOH + HNO3 = NaNO3 + H2O
  3. 2NaNO3 (разложение при нагревании) → 2NaNO2 + O2↑

Допустимы иные последовательности к NaNO3 и далее к NaNO2, если они выполнимы школьными реагентами.

Задание 21 № 1182

Дана схема превращений: Cu → Cu(NO3)2 → X → CuCl2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 6, 2025)

Cu + 4HNO3 = Cu(NO3)2 + 2NO2 + 2H2O

Cu(NO3)2 + 2NaOH = Cu(OH)2↓ + 2NaNO3
(X — Cu(OH)2)

Cu(OH)2 + 2HCl = CuCl2 + 2H2O

Задание 21 № 1205

Дана схема превращений:

H2S → X → (NH4)2SO3 → Na2SO3

Напишите уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 7, 2025)

SO2 + 2NH3 + H2O = (NH4)2SO3
(X — SO2)

(NH4)2SO3 + 2NaOH = Na2SO3 + 2NH3 + 2H2O

Задание 21 № 1067

Дана схема превращений:

Fe(NO3)2 → Fe(OH)2 → X → Fe2O3

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

  2. 4Fe(OH)2 + 2H2O + O2 = 4Fe(OH)3  (X — Fe(OH)3)
