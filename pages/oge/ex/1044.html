Задание 21 № 1044

Химические свойства простых и сложных веществ

Дана схема превращений:

Cu(NO3)2 → Cu → CuCl2 → Cu(OH)2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 1, 2025)
  1. Cu(NO3)2 + 2Ag = 2AgNO3 + Cu↓
  2. Cu + Cl2 = CuCl2
  3. CuCl2 + 2NaOH = Cu(OH)2↓ + 2NaCl

Источник: ОГЭ. Типовые экзаменационные варианты, вариант 1 (Издательство «Национальное образование», 2025).
