Задание 20 (№ 1158 · все примеры типа)

Окислительно-восстановительные реакции

Используя метод электронного баланса, расставьте коэффициенты в схеме реакции:

CO + Cl2 + KOH → K2CO3 + KCl + H2O

Укажите окислитель и восстановитель.

Эталон ответа (вариант 5, 2025)

Электронный баланс (в базной среде): C2+ − 2ē → C4+; Cl20 + 2ē → 2Cl−.

Восстановитель: C2+ (CO).
Окислитель: Cl2.

Уравнение реакции:

1 CO + 1 Cl2 + 4 KOH = 1 K2CO3 + 2 KCl + 2 H2O

Задание 21 (№ 1159 · все примеры типа)

Химические свойства простых и сложных веществ

Дана схема превращений:

Na → X → NaNO3 → NaNO2

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 5, 2025)

X = NaOH (ряд возможных превращений):

  1. 2Na + 2H2O = 2NaOH + H2↑
  2. NaOH + HNO3 = NaNO3 + H2O
  3. 2NaNO3 (разложение при нагревании) → 2NaNO2 + O2↑

Допустимы иные последовательности к NaNO3 и далее к NaNO2, если они выполнимы школьными реагентами.

Задание 22 (№ 1160 · все примеры типа)

Расчётная задача. Массовая доля, количество вещества, объём и масса

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

К 300 г раствора гидроксида натрия добавили избыток раствора сульфата меди(II), при этом образовался осадок массой 73,5 г. Вычислите массовую долю гидроксида натрия в исходном растворе. В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 5, 2025)

Уравнение:

2NaOH + CuSO4 = Cu(OH)2↓ + Na2SO4

Вычисления (по сборнику):

  • n(Cu(OH)2) = m / M = 73,5 / 98 = 0,75 моль
  • по уравнению n(NaOH) = 2 · n(Cu(OH)2) = 1,5 моль
  • m(NaOH) = n · M = 1,5 · 40 = 60 г
  • ω(NaOH) = m(NaOH) / m(р‑ра) = 60 / 300 = 0,20 (20%)
