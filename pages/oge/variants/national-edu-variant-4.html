Задание 20 (№ 1135 · все примеры типа)

Окислительно-восстановительные реакции

Используя метод электронного баланса, расставьте коэффициенты в схеме реакции:

MnSO4 + KMnO4 + H2O → MnO2 + H2SO4 + K2SO4

Определите окислитель и восстановитель.

Эталон ответа (вариант 4, 2025)

Электронный баланс (в кислой среде):

3 | Mn2+ + 2H2O − 2ē → MnO2 + 4H+
2 | MnO4− + 4H+ + 3ē → MnO2 + 2H2O

Восстановитель: Mn2+ (или MnSO4).
Окислитель: MnO4− (или KMnO4).

Уравнение реакции:

3 MnSO4 + 2 KMnO4 + 2 H2O = 5 MnO2 + 3 H2SO4 + 1 K2SO4

Задание 21 (№ 1136 · все примеры типа)

Химические свойства простых и сложных веществ

Дана схема превращений:

Si → SiO2 → X → H2SiO3

Напишите молекулярные уравнения реакций, с помощью которых можно осуществить указанные превращения.

Эталон ответа (вариант 4, 2025)

Одна из возможных цепочек: X = Na2SiO3.

  1. Si + O2 = SiO2
  2. SiO2 + 2NaOH = Na2SiO3 + H2O (сплавление или сильный нагрев с концентрированным раствором)
  3. Na2SiO3 + 2HCl = H2SiO3 ↓ + 2NaCl

Допустимы эквивалентные переходы (например, через силикат кальция и CO2 в водной среде), если они согласуются с условием.

Задание 22 (№ 1137 · все примеры типа)

Расчётная задача. Массовая доля, количество вещества, объём и масса

При проведении расчётов для всех элементов, кроме хлора, используйте целые значения Ar; Ar(Cl) = 35,5.

Рассчитайте объём (н. у.) сернистого газа, который выделится при взаимодействии 146 г 20%-ного раствора соляной кислоты с избытком раствора сульфита калия. В ответе запишите уравнение реакции, о которой идёт речь в условии задачи, и приведите все необходимые вычисления (указывайте единицы измерения искомых физических величин).

Эталон ответа (вариант 4, 2025)

Уравнение:

K2SO3 + 2HCl = 2KCl + H2O + SO2↑

Вычисления:

  • m(HCl) = m(р-ра) · ω = 146 · 0,20 = 29,2 г
  • n(HCl) = m / M = 29,2 / 36,5 = 0,80 моль
  • по уравнению n(SO2) = ½ · n(HCl) = 0,40 моль
  • V(SO2) = n · Vm = 0,40 · 22,4 = 8,96 л (н. у.)

Эталон ответа (вариант 4, 2025)

Выбранные реактивы: KOH и H2SO4 (AgNO3 с обоими растворами даёт одинаковый белый осадок AgCl и не различает соли).

Опыт 1 (реактив KOH) — для идентификации MgCl2.

MgCl2 + 2KOH = Mg(OH)2↓ + 2KCl;
Mg2+ + 2Cl− + 2K+ + 2OH− = Mg(OH)2↓ + 2K+ + 2Cl−;
Mg2+ + 2OH− = Mg(OH)2↓.

Опыт 2 (реактив H2SO4) — для идентификации BaCl2.

BaCl2 + H2SO4 = BaSO4↓ + 2HCl;
Ba2+ + 2Cl− + 2H+ + SO42− = BaSO4↓ + 2H+ + 2Cl−;
Ba2+ + SO42− = BaSO4↓.

Таблица (пример):

  • 1 — KOH: в растворе MgCl2 белый осадок Mg(OH)2; в BaCl2 заметных изменений нет.
  • 2 — H2SO4: в BaCl2 белый осадок BaSO4; в MgCl2 осадка нет.
