Задание 20

Окислительно-восстановительные реакции

Задание 20 № 1020

Используя метод электронного баланса, расставьте коэффициенты в схеме реакции

Fe(OH)3 + HI → FeI2 + I2 + H2O

Запишите в отдельной(-ых) строчке(-ах) формулы вещества/частицы окислителя и восстановителя. Укажите, какое(-ая) из этих веществ/частиц является окислителем, а какое(-ая) — восстановителем.

Эталон ответа

Электронный баланс:

Fe+3 + 1ē → Fe+2 |×2 2I−1 − 2ē → I20 |×1

Окислитель: железо в степени окисления +3 (или Fe(OH)3).
Восстановитель: иод в степени окисления −1 (или HI).

Уравнение реакции:

2Fe(OH)3 + 6HI = 2FeI2 + I2 + 6H2O

Задание 20 № 1043

Используя метод электронного баланса, расставьте коэффициенты в предложенной схеме реакции

CuS + FeCl3 → CuCl2 + FeCl2 + S

Укажите окислитель и восстановитель.

Эталон ответа (вариант 1, 2025)

Электронный баланс:

S−2 − 2ē → S0 |×1 Fe+3 + 1ē → Fe+2 |×2

Восстановитель: сера в степени окисления −2 (или CuS).
Окислитель: железо в степени окисления +3 (или FeCl3).

Уравнение реакции:

1CuS + 2FeCl3 = 1CuCl2 + 2FeCl2 + 1S

Задание 20 № 1089

Используя метод электронного баланса, расставьте коэффициенты в схеме реакции:

PH3 + HNO3 → H3PO4 + NO2 + H2O

Определите окислитель и восстановитель.

Эталон ответа (вариант 2, 2025)

Электронный баланс:

P−3 − 8ē → P+5 |×1 N+5 + 1ē → N+4 |×8

Восстановитель: PH3 (фосфор в степени окисления −3).
Окислитель: HNO3 (азот в степени окисления +5).

1PH3 + 8HNO3 = 1H3PO4 + 8NO2 + 4H2O

Задание 20 № 1066

Используя метод электронного баланса, расставьте коэффициенты в предложенной схеме реакции

HNO3 + HCl → Cl2 + NO + H2O

Укажите окислитель и восстановитель.

Эталон ответа

Электронный баланс:

N+5 + 3ē → N+2 |×2 2Cl−1 − 2ē → Cl20 |×3

Восстановитель: хлор в степени окисления −1 (или HCl).
Окислитель: азот в степени окисления +5 (или HNO3).

Уравнение реакции:

2HNO3 + 6HCl = 3Cl2 + 2NO + 4H2O
